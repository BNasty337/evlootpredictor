A Quick Guide to Getting Started

Welcome! To use this Eatventure Loot Predictor, you'll need to find your game's seed values. These seeds are a unique set of numbers that determine the contents of your chests. Don't worry, it's a simple process.

Here’s a step-by-step tutorial for Android devices:

  1. 1. On your Android device, go to your phone's file manager.I suggest using this file manager because it allows you to read hidden folders.
  3. 2. Navigate to the following directory: /sdcard/Android/data/com.hwqgrhhjfd.idlefastfood/files
  4. 3. Inside, you'll find several save files named savegame.json. Open the one with the most recent modification date with your favourite text editor, i use this one.
  6. Change the settings like that:

    GREEN Is for safety, so you cant edit anything and messup with the game

    RED has to be selected to have a better view of the file.

    BLUE is where you will search the word "seed".

  8. 4. Once the file is open, you'll see a lot of text. Search (blue square) for the word "seed" and find section that looks similar to this:

                    {"ChestId":0,"Count":0,"OpenedCount":1234,"Seed":123456789}     //Small Box
                    {"ChestId":1,"Count":1234,"OpenedCount":150,"Seed":123456789}   //Big Box
                    {"ChestId":2,"Count":1234,"OpenedCount":31,"Seed":123456789}    //Mine Box
                    {"ChestId":3,"Count":1234,"OpenedCount":42,"Seed":123456789}    //Space Box
                    {"ChestId":4,"Count":1234,"OpenedCount":76,"Seed":123456789}    //Seaport Box
                    {"ChestId":5,"Count":1234,"OpenedCount":58,"Seed":123456789}    //Middle-Ages Box
                    {"ChestId":6,"Count":1234,"OpenedCount":32,"Seed":123456789}    //Pet Box
                    {"ChestId":7,"Count":1234,"OpenedCount":14,"Seed":123456789}    //Club Box
                    {"ChestId":8,"Count":1234,"OpenedCount":14,"Seed":123456789}    //Gift Box (Christmas)
                    {"ChestId":9,"Count":1234,"OpenedCount":179,"Seed":123456789}   //Adventure (Zeus) Box
                    {"ChestId":10,"Count":1234,"OpenedCount":138,"Seed":123456789}  //Adventure (Pirate) Box
                    {"ChestId":15,"Count":1234,"OpenedCount":10,"Seed":123456789}   //Potion Box
                    
                                    ...

                    {"EggRarity":0,"Seed":123456789}    //Common Egg
                    {"EggRarity":1,"Seed":123456789}    //Rare Egg
                    {"EggRarity":2,"Seed":123456789}    //Epic Egg
                    {"EggRarity":3,"Seed":123456789}    //Legendary Egg
                    {"EggRarity":4,"Seed":123456789}    //Ultimate Egg
    (If you have never opened a chest of a type, so OpenedCount is 0, the first drop will be actually random)
  1. 5. Copy the seed number for each ChestId and EggRarity and paste it into the corresponding field on this page.
  2. 6. Once you've entered your seeds, you'll be able to simulate opening your chests and see what items you'll get!

Here’s a step-by-step tutorial for iOS devices:

  1. 1. Connect your iPhone to your computer.
  3. 2. Open iTunes (Windows) or Finder (Mac) and select your device.
  5. 3. Create a local backup of your device on the computer.
  7. 4. After the backup is completed, download iMazing, open it, go to the Backups tab, and select the backup you created.
  9. 5. Find the Apps section and wait for it to load.
  11. 6. Among all applications, find Eatventure, open the Documents folder, and there you will find all savegame files.
  13. 7. Open the most recent save file and copy all the necessary information from it.
(Note: You might need to search for the word "seed" inside the file to find the correct values.)
